## 🎮 How to Play

| Key | Action |
|-----|--------|
| `W A S D` | Move your character |
| `Tab` | Toggle Shadow World |
| `C` | Open Petal Alchemy (combine petals) |
| `1–4` | Select petal slots in combine UI |
| `Enter` | Confirm combination |
| Walk into crystals | Collect them |
| Walk into markers | Use crafted petal if you have it |

## ✨ Game Systems

- 🌸 **Petal Alchemy** — Combine petals to craft 4 unique spells
- 🌑 **Shadow World** — Flip realms with Tab to access hidden paths
- 💨 **Wind Moods** — The wind shifts between curious, playful, and sad
- 🧩 **3 Puzzle Tiers** — Bloom → Weave → Echo, each harder than the last
- 🌅 **Day/Night Cycle** — Watch the world shift through dawn, day, dusk, and night
- 🪨 **Wind Stones** — Collect 3 stones to permanently reshape the wind
- 💭 **Memory Fragments** — Discover 5 story echoes hidden across the world
- 🔇 **Silence Meter** — Restore the world to silence by solving all puzzles

## 🎨 Recipes

| Combination | Result | Effect |
|-------------|--------|--------|
| 🌸 Cherry + 🌿 Moss | Root Bridge 🌿 | Unlocks bloom markers |
| 🌸 Cherry + 💧 Dew | Rain Echo 🌧 | Unlocks weave markers |
| 🌿 Moss + 🌙 Moon | Shadow Bloom ✨ | Unlocks shadow echo markers |
| 💧 Dew + 🌙 Moon | Wind Lens 🌀 | Unlocks wind echo markers |
