# 🌸 Petals & Echoes

A cozy isometric puzzle adventure where you play as a small girl catching enchanted petals to restore a fading world.

## Game Pillars
- **Petal Physics Puzzles** — Redirect wind using stones and pond reflections
- **Reflection Mechanics** — Ponds reveal a hidden shadow world
- **No Failure State** — Cozy-first design philosophy

## Structure
- `src/character/` — Player character sprite & animation
- `src/ui/` — HUD, inventory, compass
- `src/world/` — Isometric tilemap & environment
- `src/crystals/` — Crystal echo collectibles
- `src/petals/` — Petal physics & wind system
